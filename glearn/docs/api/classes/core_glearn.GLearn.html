Class GLearn

Main GLearn

Ties together all components:

  • Data ingestion from all tools
  • Pattern mining (with Tier 1/Tier 2 escalation)
  • Proposal generation
  • Counterfactual evaluation
  • Human-in-loop approval
  • Persistent state storage (patterns, proposals, metrics)
