Class VerdictPersistenceManager

SQLite Persistence Manager for VerdictAggregator

Stores:

  • Frustration history for trend detection
  • Run records for historical analysis

SQLite is required so verdict history survives restarts and production deployments fail loudly if the durable store is unavailable.
