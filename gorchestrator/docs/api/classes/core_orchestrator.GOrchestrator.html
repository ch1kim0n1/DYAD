Class GOrchestrator

Main GOrchestrator

Ties together all components:

  • Intake & Priming
  • Configuration Sampling (with Tier 1/Tier 2 escalation)
  • Parallel Execution
  • Scoring via GMirror (with Tier 2 escalation on hard gate failures)
  • Selection
  • Persistence to GBrain
